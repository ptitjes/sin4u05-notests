## Tools

### Generate the Javadoc documentation

```bash
./gradlew javadoc
```

The Javadoc documentation gets built in `build/docs/javadoc/index.html`.

### Launch the tests

```bash
./gradlew test
```

### Launch the application

On Linux, run:
```bash
GDK_SCALE=2 ./gradlew run
```
